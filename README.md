# SafeTravel LK — Women's Safety Platform (Sri Lanka)

## Project Overview
SafeTravel LK is a web-based platform developed to help women in Sri Lanka report harassment incidents on public transport such as buses, trains, and three-wheelers. The platform provides emergency contact information, incident reporting, legal rights awareness, and transport safety guidance.

This project was developed using HTML, CSS, and JavaScript and published using GitHub Pages.

---

## Features
- Emergency hotline numbers
- SOS emergency popup
- Incident reporting form
- Complaint reference ID generator
- Transport safety information
- Legal rights information
- Public awareness statistics
- Responsive web design
- Animated user interface

---

## Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub Pages

---

## Purpose of the Project
The purpose of this platform is to:
- Encourage women to report harassment incidents
- Provide quick access to emergency contacts
- Increase awareness about women's legal rights
- Improve safety in public transportation in Sri Lanka
